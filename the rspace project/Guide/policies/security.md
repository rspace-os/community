# Security Policy

The RSpace project takes security seriously. We appreciate the security community's efforts to responsibly disclose vulnerabilities.

## Reporting a Vulnerability

**If you discover a security vulnerability, please report it responsibly following the instructions here [Security Policy](https://github.com/rspace-os/.github/blob/main/SECURITY.md)**

## Suggesting security improvements

If you have ideas or requests for making RSpace more secure, follow the contribution guidelines for contributing improvements or feature requests
