# Guide to the RSpace Project

**👋 Welcome to the RSpace open source project!** 

This is your guide to understanding and contributing to the RSpace open source project. RSpace is an open-source research data management platform for researchers, research data managers, and research infrastructure providers designed to be a user-friendly and integrated frontend for researchers to efficiently connect and utilize research infrastructure in sustainable FAIR research workflows across the entire research lifecycle.

Whether you're a researcher, developer, designer, institution, or infrastructure provider, you'll find what you need here. If you're missing any information or want to contribute to improving this guide, feel free to raise an issue or a pull request.
